# Aseets

Os assets dos alimentos e das hortas são separados em duas categorias. 

| **Categoria** |   **Descrição**    |
|----------------|:-----------|
| Cover          | Imagens usadas nas telas inciais, nas células, como um preview.  |
| Page           | Imagens usadas nas respectivas páginas dos alimentos/hortas.     |
